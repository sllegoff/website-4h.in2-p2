Rufus


Rufus is een handig programma voor het installeren van installatiemedia op bijvoorbeeld een USB-stick

Ubuntu


Ubuntu is de meest gebruikte distributie van Linux.

Linux Mint


Linux Mint is wat ik persoonlijk gebruik voor mijn apparaten. Linux mint is erg simpel en makkelijk te gebruiken, vooral als je overstapt van Windows.

Kali Linux


Kali Linux is een distributie gefocussed op cybersecurity. Het komt standaard met een heleboel hulpmiddelen voor cybersecurity.
