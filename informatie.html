Veelgestelde vragen.


Hier zal ik wat veelgestelde vragen beantwoorden over deze website. Heb je nog meer vragen? Ga dan naar de contactpagina.

Waarom deze website?


Ik heb deze website gemaakt omdat ikzelf Linux een geweldig bestuurssysteem vind tegenover alternatieven, maar er is veel te weinig aandacht voor, dus meeste mensen weten niet eens dat het bestaat. Deze website bestaat dus om meer aandacht te vragen voor Linux en ik hoop dat meer mensen het gaan gebruiken.

Hoe is deze website gemaakt?


Ik heb de website handmatig gecodeerd en ik heb gebruikt gemaakt van Visual Studio Code en GitHub. Ik heb 1 lijn aan code van Stackoverflow genomen
Deze website is gemaakt door een echt persoon.

Door wie is deze website gemaakt?


Dat kan ik beantwoorden, ook al voelt dat raar want ik heb de vraag ook moeten opschrijven. Ik ben Sam, je hebt me mischien al gezien aan de copyright onderaan de pagina. Ik heb deze website gemaakt voor een schoolopdracht. Voor de rest is er niks interessants dat jij over mij hoeft te weten.
